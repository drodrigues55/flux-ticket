# Features

> Status: Active
> Last Updated: June 2026

---

# Overview

This document summarizes the feature groups that are already present or explicitly staged in the roadmap.

It should be read as a product map, not a technical design.

---

# Shipped Feature Groups

## Consumer Portal

- event discovery
- event detail pages
- reservation-first checkout
- ticket viewing
- PDF and wallet delivery paths

## Organizer Dashboard

- real-time event ranking
- sales overview
- recent sales feed
- lot performance
- operational alerts
- operational controls

## Staff PWA

- offline validation support
- device-local sync
- operator identification
- scan confirmation flow

## Core Engine

- Redis-backed concurrency control
- outbox-based async processing
- worker-based payment recovery
- dead-letter handling

---

# Planned Feature Groups

Features that remain on the roadmap include:

- payment provider integration hardening
- richer financial reporting
- customer post-sale flows
- support tooling
- marketing and coupon systems
- predictive analytics

These items belong to future releases unless they are already explicitly implemented elsewhere in the codebase.

---

# Product Rule

If a feature cannot be traced to the current app, backend, or worker implementation, it should stay in the planned section.
