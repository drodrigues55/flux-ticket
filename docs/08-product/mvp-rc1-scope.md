# MVP RC1 Scope

Freeze scope definitions for Flux Tickets Release Candidate 1.

## Scope Freeze Details
- **Organizer Platform**: Complete event detail edits, ticket creations, validation alerts, and publishing.
- **Consumer Flow**: Event catalogue, ticket reservation, and checkout loops.
- **Staff Gate Operations**: Scan QR check-ins, manual search, offline storage download, and offline synchronization conflict detection.
- **Financial center**: Est. gross, fees, net balances, ledger lists, and CSV export boundaries.
- **Dashboard Command Center**: Priority rating engine prioritizing events and displaying Hero Card warnings.
- **Organization Management**: User scopes, roles permissions matrices, and last-owner protection guidelines.

## Explicitly Excluded Features
- Real credit card/pix gateways.
- Automatic refund logic.
- Native apps.
- Seating, coupons, and white labels.
