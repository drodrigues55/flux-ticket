# Demo Walkthrough

Guidance for repeated platform demonstrations of the Flux Tickets MVP loop.

## Walkthrough Script

### 1. Dashboard Overview
- Navigate to the Dashboard Command Center.
- Point out the Hero Card event indicating why it needs attention (e.g. low sales, approaching window).

### 2. Creation & Editing
- Click on "Criar Evento" to open the creation wizard.
- Create a test event with minimal ticket configuration, save as draft, and verify the publishing blockers checks list.

### 3. Public Catalog & Checkout
- Open the public events directory.
- Choose the newly published event.
- Complete ticket selection, reserve the ticket, fill checkout inputs, and execute mock payment.

### 4. Gate Validation
- Open the Staff PWA portal.
- Select the event and download the offline check-in registry.
- Scan the client confirmation QR code and confirm check-in approval.
